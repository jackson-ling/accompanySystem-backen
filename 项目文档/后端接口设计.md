# 陪护系统后端接口文档

> 适用于 Vue3 前端 + Spring Boot 后端

---

## 一、接口规范

### 1.1 基本规范

- **Base URL**: `http://localhost:8080/api`
- **请求格式**: JSON (`Content-Type: application/json`)
- **认证方式**: Bearer Token (Authorization Header)
- **响应格式**:
```json
{
  "code": 200,
  "message": "success",
  "data": {}
}
```

### 1.2 公共参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| page | int | 否 | 页码，默认1 |
| pageSize | int | 否 | 每页数量，默认10 |

### 1.3 响应状态码

| code | 说明 |
|------|------|
| 200 | 成功 |
| 400 | 请求参数错误 |
| 401 | 未授权/登录过期 |
| 403 | 禁止访问 |
| 404 | 资源不存在 |
| 500 | 服务器内部错误 |

## 二、认证模块 (Auth)

### 2.1 用户登录

**POST** `/auth/login`

**请求参数**:
```json
{
  "phone": "13800138000",
  "password": "123456"
}
```

**响应示例**:
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "userId": 1,
    "nickname": "张三",
    "avatar": "https://xxx.com/avatar.jpg",
    "userType": 1,
    "phone": "13800138000"
  }
}
```

### 2.2 用户注册

**POST** `/auth/register`

**请求参数**:
```json
{
  "phone": "13800138000",
  "password": "123456",
  "verifyCode": "123456",
  "nickname": "张三"
}
```

**响应**: 返回 token 和用户信息，同登录

### 2.3 退出登录

**POST** `/auth/logout`

**请求头**: 需要携带 Token

**响应**: `{ "code": 200, "message": "success" }`

### 2.4 重置密码

**POST** `/auth/reset-password`

**请求参数**:
```json
{
  "phone": "13800138000",
  "verifyCode": "123456",
  "newPassword": "123456"
}
```

**响应**: `{ "code": 200, "message": "success" }`

## 七、订单模块 (Order)

### 7.1 用户 - 创建订单

**POST** `/user/orders`

**请求参数**:
```json
{
  "serviceId": 1,
  "companionId": 2,
  "patientId": 3,
  "hospital": "北京协和医院",
  "department": "骨科",
  "appointmentTime": "2024-01-20 10:00:00",
  "pickupOption": 1,
  "pickupAddress": "",
  "remarks": "需要轮椅"
}
```

**字段说明**:
| 字段 | 类型 | 说明 |
|------|------|------|
| serviceId | long | 服务项目ID |
| companionId | long | 陪诊师ID(可选，不填则进入抢单大厅) |
| patientId | long | 就诊人ID |
| hospital | string | 医院名称 |
| department | string | 科室 |
| appointmentTime | string | 预约时间 |
| pickupOption | int | 接人方式: 1-医院门口, 2-指定地点 |
| pickupAddress | string | 接人地址(pickupOption=2时必填) |
| remarks | string | 备注 |

**响应**: `{ "code": 200, "message": "success" }`

### 7.2 用户 - 获取订单列表

**GET** `/user/orders`

**查询参数**:
| 参数名 | 类型 | 说明 |
|--------|------|------|
| status | int | 订单状态筛选 |
| page | int | 页码 |
| pageSize | int | 每页数量 |

**订单状态说明**:
| status | 说明 |
|--------|------|
| 0 | 待支付 |
| 1 | 待接单 |
| 2 | 服务中 |
| 3 | 已完成 |
| 4 | 已取消 |
| 5 | 退款中 |
| 6 | 已退款 |

**响应示例**:
```json

  "data": {

        "companionName": "李医生",
        "patientId": 3,
        "hospital": "北京协和医院",
        "department": "骨科",
        "appointmentTime": "2024-01-20 10:00:00",
        "pickupOption": 1,
        "amount": 199.00,
        "actualAmount": 199.00,
        "status": 1,
        "createTime": "2024-01-15 10:00:00"
      }
    ],
    "total": 100,
    "page": 1,
    "pageSize": 10
  }
}
```

### 7.3 用户 - 获取订单详情

**响应示例**:
